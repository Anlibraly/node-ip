# node-ip


A module for 
[Node.js](http://nodejs.org)
 convert ip to location use local data by ipip.net and qqwry data.

## Installation

You can install it using npm:

```bash
$ npm install node-ip
```

## Example

```javascript

var ip = require('node-ip');

console.log(ip.convertIp('114.114.114.114'));

```
